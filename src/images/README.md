# Images Folder Structure

This folder contains all the images for your landscaping website gallery.

## Folder Organization

- **garden/** - Garden design project images (40 total)
- **hardscape/** - Hardscaping project images (30 total)  
- **lawn/** - Lawn care project images (25 total)
- **trees/** - Tree & plant project images (25 total)
- **outdoor/** - Outdoor living poster image (1 total)
- **../videos/** - Project videos (1 total)

## Adding Your Images

1. Place your images in the appropriate folders
2. Use the naming convention: `category-##.jpg` (e.g., garden-01.jpg, hardscape-01.jpg)
3. Update the titles and descriptions in the main script.js file
4. Recommended image size: 1200px wide for best quality
5. Keep file sizes under 2MB for fast loading

## Supported Formats

- JPG, PNG, WebP for images
- MP4 for videos
